Privacy Policy

Last updated: January 1, 2025
The short version: PDFX does not collect, store, transmit, or sell any of your data — ever. Your files stay on your device. No account required. No analytics. No ads. No servers.

1. No Data Collection

PDFX is a fully offline application. We do not collect any personal information, usage data, or file contents. The app has no backend server, no database, and no internet connection requirement for its core features.

2. Your Files Stay on Your Device

All PDFs you open, edit, merge, split, or export remain entirely on your device. PDFX never uploads your files to any server, cloud service, or third party. Files are processed locally in the app's memory and saved back to your device.

  • No file content is ever transmitted over the internet
  • No cloud storage is used or required
  • Deleting the app removes all app data permanently

3. Permissions We Request

PDFX requests the following Android permissions:

  • READ_EXTERNAL_STORAGE / READ_MEDIA_DOCUMENTS — To open PDF files you select from your device storage
  • WRITE_EXTERNAL_STORAGE — To save output PDF files to your chosen location (Android 9 and below only)
  • VIBRATE — For haptic feedback on button presses
  • INTERNET — Used only for PDF rendering engine (PDF.js), loaded once when viewing a PDF. Future versions will bundle this fully offline.

We do not request location, microphone, contacts, phone state, or any other sensitive permission.

4. No Third-Party SDKs or Analytics

PDFX does not include:

  • Analytics tools (Google Analytics, Firebase, Mixpanel, etc.)
  • Crash reporting services that transmit data off-device
  • Advertising SDKs or networks
  • Social login or account systems
  • Any third-party tracking code

5. Internet Usage

PDFX uses the internet only for loading the PDF.js rendering engine on first use. This is a one-time download from the Mozilla CDN (Cloudflare). No user data is sent during this request — it is a standard content download identical to visiting a website. Future versions of PDFX will bundle the rendering engine offline, removing this requirement entirely.

6. Children's Privacy

PDFX is a productivity tool suitable for all ages. We do not knowingly collect any information from children under 13. Since we collect no information from any user at any age, this policy applies equally to all users.

7. Data Safety (Google Play)

In accordance with Google Play's Data Safety requirements:

  • Data collected: None
  • Data shared with third parties: None
  • Security practices: All data processing occurs on-device
  • Data deletion: Uninstalling the app removes all app data

8. Changes to This Policy

If we update this Privacy Policy, we will update the "Last updated" date above. Since we collect no data, any changes will only reflect feature additions or legal clarifications — never changes to data collection practices.

9. Open Source

PDFX is built on open-source technologies. The core PDF manipulation is handled by pdf-lib and PDF.js, both of which are auditable open-source projects.
